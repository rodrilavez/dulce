# Historia Clínica Dental - Dra. Dulce María Velázquez Quintana

Sistema web para gestionar historias clínicas dentales con generación automática de PDF.

## Características

- Formulario completo de historia clínica dental
- Responsive design (funciona en móvil y escritorio)
- Generación de PDF con el nombre del paciente
- Vista previa antes de descargar o imprimir
- Colores diferenciados para respuestas SI/NO
- Firma manual en PDF impreso

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- jsPDF
- html2canvas

## Despliegue

Este proyecto está listo para desplegarse en Vercel.

## Uso

1. Completa el formulario con los datos del paciente
2. Haz clic en "Generar PDF"
3. Descarga o imprime el documento
4. Firma manualmente el PDF impreso
